IOPaint - 이미지 인페인팅 도구
================================

설치 방법:
----------
1. IOPaint 폴더를 원하는 위치에 압축 해제합니다.
2. 설치는 필요 없습니다!

사용 방법:
----------
1. "run_iopaint.bat" 파일을 더블클릭합니다.
2. 잠시 기다리면 웹 브라우저가 자동으로 열립니다.
3. 이미지를 업로드하고 인페인팅을 시작하세요!

종료 방법:
----------
1. 웹 브라우저를 닫습니다.
2. 검은색 명령 프롬프트 창에서 아무 키나 누릅니다.

문제 해결:
----------
- 브라우저가 자동으로 열리지 않으면: http://localhost:8080 을 직접 열어보세요.
- 포트가 이미 사용 중이라는 오류: run_iopaint.bat 파일을 편집하여 8080을 다른 번호로 변경하세요.
- CPU 버전으로 실행되므로 GPU가 있어도 GPU를 사용하지 않습니다.

시스템 요구사항:
--------------
- Windows 10 이상
- 최소 4GB RAM 권장
- 인터넷 연결 불필요 (첫 실행 후)
